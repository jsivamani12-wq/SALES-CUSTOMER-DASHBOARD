# Sales & Customer Analytics Dashboards | Tableau

**Interactive Sales Performance & Customer Analytics Dashboards** built in Tableau using 4 years of Superstore transaction data (2020–2023). 

This project demonstrates end-to-end data analytics skills — from **data cleaning with Power Query** to building professional, insightful, and interactive dashboards.

## Project Highlights

- **Two Professional Dashboards**: Sales Performance + Customer Analytics
- **Data Period**: 2020 to 2023 (4 years of transactions)
- **Data Model**: 4 cleaned tables — **Orders, Customer, Location, Product**
- **Data Cleaning**: Thoroughly cleaned and transformed using **Power Query**
- Original YouTube tutorial + **Two custom variations** with modern dark theme and enhanced interactivity

## Dashboards Included

1. **Sales Dashboard** (2023 & 2022,2021,2020 views)
   - KPIs with YoY growth (Sales, Profit, Quantity)
   - Monthly trends with highest/lowest month indicators
   - Sub-category performance comparison
   - Weekly sales & profit trends

2. **Customer Dashboard**
   - Customer segmentation and behavior analysis
   - Sales contribution by customers
   - High-value vs low-value customer insights
   - Purchase patterns and concentration analysis

## Skills Demonstrated

- Year-over-Year (YoY) growth calculations
- Data modeling with multiple related tables
- Professional dashboard layout and UI/UX (Dark theme)
- Data cleaning & transformation using Power Query
